Youbike 站點分析預測及排程

Jun 18, 2019

2018-2 《資料科學導論》期末專題, 成績: A+

公館地區是 Youbike 租借熱區，時常會有無⾞車車可借或無地可還的問題。這個專題以台北市資料開放平臺 每⽉月 200 萬筆 Youbike 租借資料及站點地理理資料，透過 RNN 預測未來來熱⾨門站點，以最短距離優先、 Kmeans 等策略略模擬補給卡⾞車車的排程，最終以 Leaflet.js 將我們所得到的結果視覺化在地圖上。

Tools

  • pandas, numpy
  • folium [地理資料視覺化使用 folium]
  • keras
